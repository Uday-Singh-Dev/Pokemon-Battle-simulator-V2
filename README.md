# Pokemon-Battle-simulator-V2
The remake of my original Pokémon battle simulator with much more efficient code

A fully playable Pokémon-style battle game built in Python using Tkinter.

# Features
- Full type effectiveness system across 15 Pokémon types
- Animated health bars and XP gain
- Level-up system with stat increases
- Speed-based turn order
- Save and load system using file handling
- PokéCentre healing screen
- Type-coloured UI buttons

# Built with
- Python
- Tkinter
